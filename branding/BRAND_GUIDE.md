# ForgeDesk Brand Guide

## Brand Philosophy

ForgeDesk stands for **crafted power with discipline**.

- **Forge** = build with strength, precision, and repeatability.
- **Desk** = developer workspace, practical productivity, and everyday shipping.

The visual identity combines a forged frame (trust + structure) with a spark core (innovation + momentum).

## Voice

- Clear, direct, technical
- Confident but never noisy
- Security-first and developer-respectful

## Design Principles

1. **Strong foundation**: stable systems, explicit contracts, predictable behavior.
2. **Fast path to shipping**: short feedback loops, simple commands, clear docs.
3. **Safe by default**: capability boundaries and production discipline.

## Logo System

- Primary logo: `branding/forgedesk-logo.svg`
- Mark/icon: `branding/forgedesk-mark.svg`

Use the full logo in README headers, website hero sections, and release notes.
Use the mark for avatars, app icons, and compact contexts.
